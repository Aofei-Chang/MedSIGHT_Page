Place your tutorial video here.

Required filename:
    tutorial.mp4

Optional poster image (shown before the video plays):
    poster.jpg

The <video> tag in ../../index.html (in the "Video Tutorial" section) is
already wired up to load these two files automatically.

Recommended:
- Keep the file size under ~50 MB so GitHub Pages serves it quickly.
- 1280x720 or 1920x1080, H.264 / AAC, MP4 container.

If the file is missing, the page shows a "Video coming soon" placeholder
instead of a broken player.
